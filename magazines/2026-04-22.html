SUPPLY CHAIN BREACH

Vercel breach: OAuth attack exposes risk in platform environment variables

Directly applies to you

Trend Micro publishes forensic analysis of the Vercel incident, showing how attackers chained an OAuth token compromise into broad access to environment variables across customer projects.

The analysis traces the attack path from a compromised integration token through token minting and lateral movement into customer workspaces. Secrets stored as platform environment variables (API keys, database credentials, signing keys) were exfiltrated at scale. The key takeaway is that OAuth scopes granted to platforms become a blast radius of their own once any part of that platform is breached.

The authors call out the pattern explicitly: any CI/CD or hosting platform that holds long-lived OAuth tokens across tenants is a single point of failure for every project it touches. This is the third major supply chain incident in 2026 involving a hosting platform and stored secrets.

What to do

Rotate any secrets stored in Vercel environment variables. Audit OAuth apps across GitHub and other dev platforms for token scopes, and revoke anything over-privileged. Move high-value secrets to a dedicated vault with short-lived credentials and treat hosting platform env vars as compromised-by-default.

SURVEILLANCE

Meta to start capturing employee mouse movements and keystrokes for AI training

Directly applies to you

Reuters reports Meta will collect fine-grained behavioural telemetry from its own staff (cursor trails, keystroke timing, app focus) to train internal AI models, triggering immediate questions under EU works-council law.

The scheme covers internal productivity tools and applies to employees in multiple regions. Works councils in Germany and Ireland have already signalled challenges under local co-determination and GDPR Article 88 rules on processing of employee data. The UK ICO has a similar test on necessity and proportionality.

Expect this to become the reference case employees and regulators cite whenever a company proposes AI-from-the-inside programmes. For anyone running security or HR tech stacks in Europe, it is also a preview of the kind of monitoring vendors will start pitching as an upsell to existing DLP and insider-threat products.

What to do

If your organisation operates in the UK or EU, check whether any current or planned telemetry (DLP, endpoint agents, productivity analytics) could be reclassified as AI training data under a vendor update. Get Legal and DPO in front of the decision rather than behind it, and set a policy position before procurement pushes a renewal.

PLATFORM CHANGE

Claude Code being removed from Anthropic's Pro plan?

Directly applies to you

A post from Ed Zitron claims Claude Code will be pulled from the Claude Pro tier, pushing coding use into higher-priced plans. Anthropic has not yet confirmed or denied.

The claim is sourced to customer-facing documentation changes rather than an official announcement, so treat the specifics with caution. Either way, if accurate it resets the cost model for individuals and small teams who have been building Claude Code into daily workflows on the Pro plan.

Anthropic's public position on Claude Code has been that it is a flagship product for serious coding work, which is consistent with pushing it into Max or the developer API billing rather than bundling it cheaply. Expect a formal announcement inside the next week.

What to do

Before any plan change lands, export any Cowork sessions, skills, and CLAUDE.md files you care about. Model what your Claude Code usage would cost on Max or API billing and decide in advance whether to upgrade, switch to API, or rebuild equivalent workflows on another runtime. Do not wait for the migration email.

Changes to GitHub Copilot individual plans

GitHub has announced changes to Copilot individual plans, restructuring the free tier and adjusting model access on paid plans.

The post lays out new quotas on agent mode, Chat, and premium models, plus a refreshed pricing table for Copilot Pro and Pro+. GitHub frames the shift as a move away from unlimited usage on frontier models toward metered consumption, following the same direction the rest of the market has taken through 2026.

If your team standardised on Copilot under the previous plan, the practical effect is that your heaviest users will hit limits sooner and the cost per seat effectively rises. The administrative knobs for org admins have also been tightened.

AGENT SECURITY RESEARCH

CrabTrap: An LLM-as-a-judge HTTP proxy to secure agents in production

Directly applies to you

Brex has open-sourced CrabTrap, an HTTP proxy that places a secondary LLM between production agents and the wider network, intercepting tool calls for policy review before they execute.

The design treats the agent as untrusted and the proxy as the control plane. Each outbound request carries enough context for a judging model to apply policy, strip secrets, redact PII, and log a structured audit trail. Latency overhead is significant but bounded, and the authors argue the trade is worth it for any agent that can write or call money-moving APIs.

CrabTrap is not a silver bullet. A judging LLM is itself attackable, and the real value is in making policy explicit and reviewable, not in catching every adversarial prompt. It is a useful reference architecture for anyone who has an agent programme moving past prototype.

What to do

If you are standing up an agent platform, review CrabTrap as a pattern for enforcing policy at the network boundary rather than inside each agent. Assess whether your current agent deployments log tool calls in a way a security team can actually audit after an incident.

NEW MODEL

ChatGPT Images 2.0

OpenAI has launched ChatGPT Images 2.0, a refreshed image generation and editing system inside ChatGPT with markedly better text rendering, instruction following, and multi-image composition.

The update sits inside the standard ChatGPT experience rather than as a separate product, and covers generation, editing, and style-preserving variations. The biggest practical jump is in handling text inside images (posters, UI mockups, diagrams), which has been a persistent weakness of earlier systems.

Expected second-order effects include more convincing phishing lures and more realistic fabricated documents. Anyone still treating image metadata or visual fidelity as a signal of authenticity needs to revisit that assumption.

07
RETAIL INCIDENT

Edit store price tags using Flipper Zero

A new Flipper Zero app, TagTinker, demonstrates rewriting the electronic shelf label (ESL) tags used by major retailers, with minimal access and no authentication on many deployed systems.

The research builds on years of work showing that ESL systems rely on proprietary radio protocols with weak or absent cryptography. TagTinker packages the attack into a one-button app, which is what turns a lab curiosity into something that will inevitably appear in a news story within weeks.

From a security perspective, the interesting angle is not the prank value but the pattern: retail operational technology continues to ship without authenticated firmware and without a credible update path. The same category of problem sits behind payment terminals, digital signage, and a large chunk of store-floor IoT.

DEV TOOL

Ctx: a /resume that works across Claude Code and Codex

Directly applies to you

A small open-source CLI that snapshots project context and restores it across different agentic coding tools, so a session started in Claude Code can be handed to Codex (and back) without losing working memory.

Ctx writes a portable file containing task state, relevant file paths, recent decisions, and planned next steps, and loads it back into whichever tool you resume in. It sidesteps the vendor-specific session formats of each tool by treating context as a first-class artefact that lives in the repo.

For anyone who runs multiple agents in parallel or hot-swaps between them depending on task type, this is a practical answer to the current fragmentation. It also doubles as a simple continuity mechanism if a provider changes pricing or availability at short notice.

What to do

Try Ctx on one active Claude Code project and see whether the exported context actually carries your working state. If it does, make it part of your default project scaffolding so you are never locked to a single agent runtime.

09
GOVERNANCE

Less human AI agents, please

An argument against dressing agents in human names, faces, and first-person voice, on the grounds that it degrades user judgement about when to trust outputs and muddies accountability when things go wrong.

The piece is short and blunt: the more human an agent looks and sounds, the less likely users are to apply the scepticism they would to a tool, and the more likely they are to treat its mistakes as personal rather than systemic. The author argues for visible seams, named tools, and explicit confidence signalling instead.

For CISOs and product leaders, this is a useful counter-pressure to the commercial instinct to make every assistant feel like a colleague. Policy, training, and incident response all work better when the organisation is clear that an agent is software and its outputs are artefacts, not opinions.

INVESTIGATION

FBI looks into dead or missing scientists tied to NASA, Blue Origin, SpaceX

Fortune reports the FBI is reviewing a cluster of deaths and disappearances of researchers employed across major US space and aerospace programmes, at the request of several of the affected families.

The story reads carefully, avoids sensationalism, and notes that no pattern has been established. What is documented is a list of individuals, their employers, and the circumstances of each case as currently known to investigators. Several families say earlier enquiries produced unsatisfactory answers and want the Bureau to take a wider view.

There is no claim of foul play on the record. The piece is worth reading as a reminder that high-consequence industries carry real personal risk, and that the people behind headline technologies are often working under pressure that does not show up in press releases.
